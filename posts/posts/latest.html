Browser comparison!

Who doesn’t like little bit of benchmarking? I sure do whenever it’s a GPU, CPU, Smartphone or whatever even bootimes. But last week I started comparing Web Browsers, just so you understand I wasn’t benchmarking how taxing are they or OS differences. All test was done on Windows 10 1609 on my Acer Aspire V17 nitro.

After figuring out that not all browsers were running on the right GPU even though I set Nvidia to be the default one I ran all the test multiple times sometimes with some programs in background. All browsers got the same chances. Here are the results! If you want to test your browser links to the benchmarks are on the bottom of this post!

Google’s Chrome got the first place with Opera just 1% behind on the second place and on

The third place was Firefox which surprised me by winning the maximum best in the peacekeeper benchmark and the best css3 support, which might be right but still Firefox in pain in the ass for my webpage development you can always see some warning not working on Firefox.

 On the fourth place was Vivaldi which felt like a total copy of Opera to me the browsers are so similar I guess they share some code

Fifth place got the Maxthon Cloud browser which I might spend some more time in because it looks like a good cloud browser even though it doesn’t support WebGL 2.0

Sixth place got UC the Chinese browser I didn’t liked this one to be honest even though this one supports WebGL 2.0 it got beaten by Maxthon so who cares.

On the seventh place was finally the all new Microsoft EDGE to be honest I was expecting more I use this browser on my windows tablet and the performance on 1GB an Intel Atom CPU is amazing but this was just a feature score test so not that good results.

Last is Internet Explorer which still exists on Windows 10 why to be honest, I know some core windows components are tied to Internet explorer but really Microsoft kill this one. It wasn’t even able to run the last test after hitting the run button it just redirects and page not found that’s it.
